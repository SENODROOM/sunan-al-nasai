# CLI Reference — sunan-al-nasai

```
nasai <hadithId>
nasai <chapterId> <hadithId>
nasai --search "<query>" [--all]
nasai --chapter <id>
nasai --random
nasai --react
nasai --version
nasai --help
```

## Language flags
- (default) — English only
- `-a` / `--arabic` — Arabic only
- `-b` / `--both` — Arabic + English
