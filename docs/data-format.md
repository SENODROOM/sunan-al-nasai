# Data Format — nasai.json

```json
{
  "metadata": {
    "id": 1, "length": 5768,
    "arabic":  { "title": "السنن الصغرى", "author": "الإمام أحمد بن شعيب النسائي" },
    "english": { "title": "Sunan al-Nasa'i", "author": "Imam Ahmad ibn Shu'ayb al-Nasa'i" }
  },
  "chapters": [ { "id": 1, "arabic": "...", "english": "The Book of Purification" } ],
  "hadiths": [
    { "id": 1, "chapterId": 1, "arabic": "...", "english": { "narrator": "...", "text": "..." } }
  ]
}
```

`data/chapters/` is generated by `node scripts/build.mjs`. Do not edit manually.
