# Changelog

## [1.0.0] — 2026-03-22

### Added
- Initial release
- npm package: `npm install sunan-al-nasai`
- Python package: `pip install sunan-al-nasai`
- 5,768 hadiths with full Arabic text and English translations
- Offline-first — data bundled, no CDN required
- CLI: `nasai <id>`, `--search`, `--random`, `--chapter`, `--react`
- React hook generator: `nasai --react`
- TypeScript definitions included
