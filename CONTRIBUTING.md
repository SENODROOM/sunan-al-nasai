# Contributing

## Getting Started

```bash
git clone https://github.com/SENODROOM/sunan-al-nasai.git
cd sunan-al-nasai
```

## Making Changes

- JS changes → edit files in `src/`
- Python changes → edit files in `python/sunan_al_nasai/`
- Data changes → edit `data/nasai.json` then run `node scripts/build.mjs`

## Publishing

Publishing is automatic via GitHub Actions on every GitHub Release.
